# React Shopify

React Shopify is a component library that aims to make dead simple to create custom shopify stores.

## Install

```bash
yarn add -E react-shopify graphql graphql-tag apollo-boost react-apollo
```

```bash
npm i -E react-shopify graphql graphql-tag apollo-boost react-apollo
```

React Shopify requires React 16.3+ as it uses new React Context API.
